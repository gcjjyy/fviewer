# fviewer
DOS Font viewer (4096 bytes English font, over 11520 Korean font)

# Prerequsites
Mac OS
```sh
$ brew install sdl2
```

Ubuntu
```sh
$ sudo apt install libsdl2-dev
```

# Build
```sh
$ g++ -o fviewer fviewer.cpp -lSDL2
```

# Usage
```sh
$ ./fviewer H01.HAN
```
